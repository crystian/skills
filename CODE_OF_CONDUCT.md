# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its code of conduct.

The full text is available at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — through issues, pull requests, discussions, or any other channel — you agree to abide by its terms.

## Scope

This code applies within all project spaces (issues, pull requests, discussions, commit messages, documentation) and whenever an individual is representing the project in public spaces.

## Reporting

To report an incident, contact the maintainer at:

- **Email**: infomakersia at gmail dot com — subject `[conduct] crystools-skills: <short description>`

All reports will be handled with discretion. The maintainer is responsible for clarifying and enforcing the standards described in the Covenant and will take appropriate and fair corrective action in response to any behavior deemed inappropriate.
